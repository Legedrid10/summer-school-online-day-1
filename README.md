# summer-school-online-day-1 HTML Elements Used:
- h1 to h6: For different heading levels.
- ul/ol: To list skills and education.
- nav: For page navigation.
- img: To add an image with alt text.
- table: To show project info.
- form: To collect user data in contact.html.
- input types: text, email, tel, date, range used for proper input.

Why These Inputs:
- email, tel: Validate format
- checkboxes & radio: Allow user choices
- date/range/select: For extra form enhancements

Navigation:
- Every page has links to other pages.
- Internal links: between portfolio.html, about.html, contact.html
- External link example: Google (in about.html)
